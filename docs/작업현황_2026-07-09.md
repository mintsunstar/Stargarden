## 6. 프론트엔드 구조

```
src/
├── components/     TabBar, BottomSheet, OhangBadge, Toast, DevModeBadge
├── features/
│   ├── auth/       스플래시, 로그인, 온보딩, 기질결과

│   ├── collection/ 도감, 식물상세
│   └── my/         마이페이지
├── stores/         auth, garden, seeds, sound, collection, tutorial, toast
├── lib/            growth, ohang, careActions, soundEngine, sleep, timeOfDay, supabase
├── constants/      game, seeds, sounds, strings
└── hooks/          useGardenSound, useGardenTick
```

## 7. 로컬 실행 방법

```bash
cd c:\Dev\Stargarden
npm install
npm run dev
# → http://localhost:5173/
```

- **개발자 모드**: `npm run dev` 시 자동 활성 (로그인 생략)
- **끄기**: `.env`에 `VITE_DEV_BYPASS_AUTH=false`
- **Supabase 없이** 프론트만 동작 (로컬 Zustand persist)
